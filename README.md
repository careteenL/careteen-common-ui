## 核心文件
- main.scss      (入口文件，引入所有scss文件)
- global.scss    (全局样式)
- _reset.scss    (样式归零)
- _base.scss     (基础变量：常用颜色、字体...)
- _mixin.scss    (功能样式，包括：mixin(混合)、%(placeholder选择器)、function(函数) 三个部分)
